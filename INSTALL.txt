# prepopulate.module installation instructions
---

1. Copy the prepopulate/ directory into your Drupal installation's
modules/ directory.

2. Enable the prepopulate module in admin/modules

3. For each content type that you want to allow prepopulating, set the
prepopulate checkbox in admin/settings/content-types
